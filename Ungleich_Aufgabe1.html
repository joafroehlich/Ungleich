Aufgabe 1

Bestimmen Sie die Lösungsmenge der 
Ungleichungen für x ∈ ℚ: 
 
a) x + 5 < 9 
 
b) 4 + x ≤ 7 
 
c) x - 5 > 2 
 
d) 15 ≥ x + 7 
 
e) 8 < 9 - x 
 
f) 3 ≤ x - 6    
 
 
a) x + 5 < 9 |-5 
 
    x < 4        x ∈ ]-∞ ; 4[ 
 
b) 4 + x ≤ 7 |-4 
 
    x ≤ 3        x ∈ ]-∞ ; 3] 
 
c) x - 5 > 2 |+5 
 
    x > 7        x ∈ ]7; ∞[ 
 
d) 15 ≥ x + 7 |-7 
 
    x ≤ 8        x ∈ ]-∞ ; 8] 
 
e) 8 < 9 - x |+x - 8 
 
    x < 1        x ∈ ]-∞ ; 1[ 
 
f) 3 ≤ x - 6 |+6 
 
   x ≥ 9         x ∈ [9 ; ∞[

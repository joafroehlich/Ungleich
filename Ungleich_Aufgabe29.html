Bestimmen Sie die Lösungsmenge der
Ungleichung für x ∈ ℝ: 

  3      3 
|---x + ---| ≥ |x - 2| 
  2      4 
  
Fallunterscheidung: 

1. Fall 

  3      3  
|---x + ---| = 1,5x + 0,75 für 1,5x + 0,75 ≥ 0 --> x ≥ -0,5  
  2      4        

und 

|x - 2| = x - 2 für x - 2 ≥ 0 --> x ≥ 2 

D1 = x ≥ -0,5 ∩ x ≥ 2 = x ≥ 2 

1,5x + 0,75 ≥ x - 2 |-x - 0,75 

0,5x ≥ -2,75  |:0,5 

x ≥ -5,5  

L1 = D1 ∩ x ≥ -5,5 = x ≥ 2 ∩ x ≥ -5,5 = x ≥ 2  

2. Fall 

  3      3  
|---x + ---| = 1,5x + 0,75 für 1,5x + 0,75 ≥ 0 --> x ≥ -0,5  
  2      4    

und  

|x - 2| = -(x - 2) für x - 2 < 0 --> x < 2 

D2 = x ≥ -0,5 ∩ x < 2 = -0,5 ≤ x < 2 

1,5x + 0,75 ≥ -(x - 2)  

1,5x + 0,75 ≥ -x + 2 |+x - 0,75 

2,5x ≥ 1,25  |:2,5 

x ≥ 0,5  

L2 = D2 ∩ x ≥ 0,5 = x < 2 ∩ x ≥ 0,5 = 0,5 ≤ x < 2  

3. Fall 

  3      3  
|---x + ---| = -(1,5x + 0,75) für 1,5x + 0,75 < 0 --> x < -0,5  
  2      4

und  

|x - 2| = x - 2 für x - 2 ≥ 0 --> x ≥ 2 

D3 = x < -0,5 ∩ x ≥ 2 = ∅ 

-(1,5x + 0,75) ≥ x - 2  

-1,5x - 0,75 ≥ x - 2 |+1,5x + 2 

2,5x ≤ 1,25  |:2,5 

x ≤ 0,5  

L3 = D3 ∩ x ≤ 0,5 = ∅ ∩ x ≤ 0,5 = x ≤ 0,5 

4. Fall 

  3      3  
|---x + ---| = -(1,5x + 0,75) für 1,5x + 0,75 < 0 --> x < -0,5  
  2      4

und  

|x - 2| = -(x - 2) für x - 2 < 0 --> x < 2 

D4 = x < -0,5 ∩ x < 2 = x < -0,5 

-(1,5x + 0,75) ≥ -(x - 2)  

-1,5x - 0,75 ≥ -x + 2 |+1,5x - 2 

0,5x ≤ - 2,75 |:0,5 

x ≤ -5,5  

L4 = D4 ∩ x ≤ -5,5 = x < -0,5 ∩ x ≤ -5,5 = x ≤ -5,5 

L = L1 ∪ L2 ∪ L3 ∪ L4 = x ≥ 2 ∪ 0,5 ≤ x < 2 ∪ x ≤ 0,5 ∪ x ≤ -5,5   

L = x ≥ 0,5 ∪ x ≤ -5,5

Bestimmen Sie die Lösungsmengen der Ungleichungen für x ∈ ℝ: 

a) |x| > -2 
b) |x - 4| > 2 
c) |5 + x| < 3 

a)  

|x| > -2  Der Betrag von x ist für alle x immer größer als 0, 

also positiv  --> |x| > -2 für alle x 

L = ]-∞ < x < ∞[  

b)  

|x - 4| > 2 

Fallunterscheidung: 

1. Fall |x - 4| = x - 4 für x - 4 ≥ 0 --> x ≥ 4 

x - 4 > 2 |+4 

x > 6

L1 = x ≥ 4 ∩ x > 6 = x > 6 

2. Fall |x - 4| = -(x - 4) für x - 4 < 0 --> x < 4 - (x - 4) > 2 

-x + 4 > 2 |+x - 2 

x < 2 

L2 = x < 4 ∩ x < 2 = x < 2 

L = L1 ∪ L2 

L = x > 6 ∪ x < 2    

c) 

|5 + x| < 3 

Fallunterscheidung:

1. Fall |5 + x| = 5 + x für 5 + x ≥ 0 --> x ≥ -5 

5 + x < 3 |-5 

x < -2 

L1 = x ≥ -5 ∩ x < -2 = -5 ≤ x < -2 

2. Fall |5 + x| = -(5 + x) für 5 + x < 0 --> 

x < -5 - (5 + x) < 3 

- 5 - x < 3 |+x - 3 

x > -8 

L2 = x < -5 ∩ x > -8 = -8 < x < -5 

L = L1 ∪ L2 = -5 ≤ x < -2 ∪ -8 < x < -5 

L = -8 < x < -2

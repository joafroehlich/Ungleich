Ungleichung für x ∈ ℝ: 

(x + 4)² ≥ 1  

(x + 4)² ≥ 1 |ⱱ 

|x + 4| ≥ 1 

Fallunterscheidung: 

1. Fall 

|x + 4| = x + 4 für x + 4 ≥ 0 --> x ≥ -4 

x + 4 ≥ 1 |-4 

x ≥ -3 

L1 = x ≥ -4 ∩ x ≥ -3 = x ≥ -3 

2. Fall 

|x + 4| = -(x + 4) für x + 4 < 0 --> x < -4 

-(x + 4) ≥ 1  

-x - 4 ≥ 1 |+x - 1 

x ≤ -5 

L2 = x < -4 ∩ x ≤ -5 = x ≤ -5 

L = L1 ∪ L2 = x ≥ -3 ∪ x ≤ -5

Aufgabe 19

Bestimmen Sie die Lösungsmenge der
Ungleichung für x ∈ ℝ: 

 x - 3     x + 1     x² - 5x - 1 
------- + ------- ≤ -------------         
 x + 4     x - 1     x² + 3x - 4 

x² + 3x - 4 = (x + 4)(x - 1) 

x ≠ - 4, x ≠ 1, weil sonst der Nenner x + 4 = 0, 
oder x - 1 = 0. Division durch 0 nicht 
definiert in ℝ.

Umformen: 
 
 x - 3     x + 1     x² - 5x – 1      x² - 5x - 1 
------- + ------- ≤ ------------- |- -------------          
 x + 4     x - 1     x² + 3x – 4      x² + 3x - 4 
 
 
  x - 3     x + 1     x² - 5x - 1 
-------- + ------- - ------------- ≤ 0         
  x + 4     x - 1     x² + 3x - 4 
 
  (x – 3)(x – 1) + (x + 1)(x + 4) – (x² - 5x – 1) 
-------------------------------------------------- ≤ 0         
                     x² + 3x – 4 
 
  x² - 4x + 3 + x² + 5x + 4 – x² + 5x + 1 
------------------------------------------- ≤ 0         
                 x² + 3x – 4 
 
  x² + 6x + 8      (x + 4)(x + 2)      x + 2 
--------------- = ----------------- = ------- ≤ 0         
  x² + 3x – 4      (x + 4)(x – 1)      x - 1 
 
Fallunterscheidung: 
 
  (x + 2) 
---------- ≤ 0,  
  (x – 1) 
 
Ist dann der Fall, wenn  
 
x + 2 ≥ 0 -→ x ≥ -2 und 
                               D1 = -2 ≤ x < 1 
x – 1 < 0 -→ x < 1 
 
oder wenn  
 
x + 2 ≤ 0 -→ x ≤ -2 und 
                               D2 = ∅ 
x – 1 > 0 -→ x > 1 
 
L = D1 ∪ D2 = -2 ≤ x < 1

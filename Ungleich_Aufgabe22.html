Bestimmen Sie die Lösungsmenge der
Ungleichung für x ∈ ℝ: 

 x² + 17     x - 2        5    
--------- ≤ -------- - -------             
 x² - 1      x + 1      x - 1 
 
     x² + 17        x - 2        5    
---------------- ≤ -------- - -------             
 (x - 1)(x + 1)     x + 1      x - 1 

x ≠ 1; - 1, weil sonst der Nenner x - 1 = 0, 
oder x + 1 = 0. Division durch 0 nicht 
definiert in ℝ.
 
(x - 1)(x + 1) = Hauptnenner 
 
Fallunterscheidung: 
 
1. Fall (x - 1)(x + 1) ist größer 0, wenn  
 
(x - 1) > 0 --> x > 1 
(x + 1) > 0 --> x > -1 
 
x > 1 ∩ x > -1 = 1. Definitionsmenge für den 1. Fall D1 = 1 < x < ∞   
 
(x - 1) < 0 --> x < 1 
(x + 1) < 0 --> x < -1 
 
x < 1 ∩ x < - 1 = 2. Definitionsmenge für den 1. Fall D2 = -∞ < x < -1 
 
D1,2 = D1 ∪ D2 = 1 < x < ∞ ∪ -∞ < x < -1  
 
    x² + 17          x - 2       5    
---------------- ≤ -------- - ------- |*(x - 1)(x + 1)            
 (x - 1)(x + 1)      x + 1     x - 1 
 
x² + 17 ≤ (x - 2)(x - 1) - 5 (x + 1) 
 
x² + 17 ≤ x² - 3x + 2 - 5x - 5 |-x² 
 
17 ≤ - 8x - 3 |+8x - 17 
 
8x ≤ -20 |:8 
 
x ≤ -2,5 
 
L1 = D1,2 ∩ x ≤ -2,5 = 1 < x < ∞ ∪ - ∞ < x < -1 ∩ x ≤ - 2,5 = x ≤ -2,5 
 
2. Fall (x - 1)(x + 1) ist kleiner 0, wenn  
 
(x - 1) > 0 --> x > 1 

(x + 1) < 0 --> x < -1 

x > 1 ∩ x < -1 = 1. Definitionsmenge für den 2. Fall D3 = ∅ 

(x - 1) < 0 --> x < 1 

(x + 1) > 0 --> x > -1 

x < 1 ∩ x > - 1 = 2. Definitionsmenge für den 2. Fall D4 = -1 < x < 1 

D3,4 = ∅ ∩ -1 < x < 1 = -1 < x < 1 

     x² + 17        x - 2        5    
---------------- ≤ -------- - ------- | *(x - 1)(x + 1) aus ≤ wird ≥           
 (x - 1)(x + 1)     x + 1     x - 1 

x² + 17 ≥ (x - 2)(x - 1) - 5 (x + 1) 

x² + 17 ≥ x² - 3x + 2 - 5x - 5 |-x² 

17 ≥ - 8x - 3 |+8x - 17 

8x ≥ -20 |:8 

x ≥ -2,5 

L2 = D3,4 ∩ x ≥ -2,5 = -1 < x < 1 ∩ x ≥ -2,5 = -1 < x < 1 

L = L1 ∪ L2

L = x ≤ -2,5 ∪ -1 < x < 1

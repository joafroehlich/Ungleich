x² + 13x + 30 > 0 

Quadratische Ergänzung:

x² + 13x + 42,25 - 42,25 + 30 > 0 

(x + 6,5)² - 12,25 > 0 |+12,25 

(x + 6,5)² > 12,25 |ⱱ 

|x + 6,5| > 3,5 

Fallunterscheidung: 

1. Fall 

|x + 6,5| = x + 6,5 für x + 6,5 ≥ 0 --> x ≥ -6,5 

x + 6,5 > 3,5 |-6,5 

x > -3

2. Fall 

|x + 6,5| = -(x + 6,5) für x + 6,5 < 0 --> x < -6,5 

-(x + 6,5) > 3,5  

-x - 6,5 > 3,5 |+x - 3,5 

x < -10 

L2 = x < -6,5 ∩ x < -10 = x < -10

L = L1 ∪ L2 = x > -3 ∪ x < -10 = 

L = x > -3 ∪ x < -10

Bestimmen Sie die Lösungsmenge der
Ungleichung für x ∈ ℝ: 

2x² + 6x - 8 < 0 

2(x² + 3x - 4) < 0

2(x + 4)(x - 1) < 0 

(x + 4)(x - 1) ist kleiner 0, wenn 

x + 4 > 0 --> x > -4 

x - 1 < 0 --> x < 1  

L1 = x > -4 ∩ x < 1 = - 4 < x < 1 

x + 4 < 0 --> x < -4 

x - 1 > 0 --> x > 1 

L2 = x < -4 ∩ x > 1 = ∅ 

L = L1 ∪ L2 = - 4 < x < 1 ∪ ∅ 

L = -4 < x < 1

Bestimmen Sie die Lösungsmenge der
Ungleichung für x ∈ ℝ: 

     1          1        
--------- - --------- ≤ 0     
  2x - 4     3x + 9  

     1           1        
---------- - ---------- ≤ 0 --> Hauptnenner 6(x - 2)(x + 3)  
2(x - 2)     3(x + 3)

x ≠ 2; -3, weil sonst die Nenner 2(x - 2) = 0, 
oder 3(x + 3) = 0. Division durch 0 nicht 
definiert in ℝ. 
   
Fallunterscheidung: 

1. Fall 6(x - 2)(x + 3) ist größer 0, wenn 

(x - 2) > 0 --> x > 2 

(x + 3) > 0 --> x > -3 

x > 2 ∩ x > -3 = 1. Definitionsmenge für den 1. Fall D1 = 2 < x < ∞  

oder wenn

(x - 2) < 0 --> x < 2 

(x + 3) < 0 --> x < -3 

x < 2 ∩ x < -3 = 2. Definitionsmenge für den 1. Fall D1 = -∞ < x < -3 

D1,2 = D1 ∪ D2 = 2 < x < ∞ ∪ -∞ < x < -3  

    1          1        
-------- - -------- ≤ 0  |*6(x - 2)(x + 3)    
 2x - 4     3x + 9  

3(x + 3) - 2(x - 2) ≤ 0 

3x + 9 - 2x + 4 ≤ 0 |-13 

x ≤ -13 

L1 = D1,2 ∩ x ≤ -13 = 2 < x < ∞ ∪ -∞ < x < -3 ∩ x ≤ -13 = x ≤ -13 

2. Fall 6(x - 2)(x + 3) ist kleiner 0, wenn 

(x - 2) > 0 --> x > 2 

(x + 3) < 0 --> x < -3 

x > 2 ∩ x < -3 = 1. Definitionsmenge für den 2. Fall D3 = ∅   

(x - 2) < 0 --> x < 2 

(x + 3) > 0 --> x > -3 

x < 2 ∩ x > -3 = 2. Definitionsmenge für den 1. Fall D4 = -3 < x < 2 

D3,4 = D3 ∪ D4 = ∅  ∪ -3 < x < 2 = -3 < x < 2  

    1           1        
--------- - --------- ≤ 0  |*6(x - 2)(x + 3) aus ≤ wird ≥   
 2x - 4      3x + 9  

3(x + 3) - 2(x - 2) ≥ 0 

3x + 9 - 2x + 4 ≥ 0 |-13 

x ≥ -13 

L2 = D3,4 ∩ x ≥ - 13 = -3 < x < 2 ∪ x ≥ -13 = -3 < x < 2 

L = L1 ∪ L2

L = x ≤ -13 ∪ -3 < x < 2 = x ≤ - 13 ∨ -3 < x < 2

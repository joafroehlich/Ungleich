x² - 6x + 10 > 0

x² + 6x + 9 - 9 + 10 > 0 

(x + 3)² + 1 > 0 |-1 

(x + 3)² > -1 

(x + 3)² ist > 0 für alle x --> L = ℝ

|x - 0,5| > 0 

Fallunterscheidung: 

1. Fall

L1 = x > 0,5  

2. Fall

x < 0,5

Bestimmen Sie die Lösungsmenge der
Ungleichung für x ∈ ℝ: 

 10 + x      x + 4         x + 3    
--------- - ------- ≤ 1 - -------      
   24x        12x            8x 
   
x ≠ 0, weil sonst die Nenner 24x = 0, 12x = 0
und 8x = 0. Division durch 0 nicht 
definiert in ℝ.
 
Fallunterscheidung, Hauptnenner 24x: 
 
1. Fall 24x > 0 --> x > 0 
 
  10 + x      x + 4         x + 3    
---------- - ------- ≤ 1 - ------- | *24x 
    24x        12x            8x    
 
10 + x - 2(x + 4) ≤ 24x - 3(x + 3) 
 
10 + x - 2x - 8 ≤ 24x - 3x - 9 |+x + 9 
 
22x ≥ 11 |:22 
 
x ≥ 0,5 
 
L1 = x > 0 ∩ x ≥ 0,5 = x ≥ 0,5 
 
2. Fall 24x < 0 --> x < 0 
 
  10 + x       x + 4           x + 3    
---------- - -------- ≤ 1 - -------- | *24x aus ≤ wird ≥
    24x         12x                8x    
 
10 + x - 2(x + 4) ≥ 24x - 3(x + 3) 
 
10 + x - 2x - 8 ≥ 24x - 3x - 9 |+x + 9 
 
22x ≤ 11 |:22 
 
x ≤ 0,5 
 
L2 = x < 0 ∩ x ≤ 0,5 = x < 0 

L = L1 ∪ L2
 
L = x ≥ 0,5 ∪ x < 0 = x ≥ 0,5 ∪ x < 0

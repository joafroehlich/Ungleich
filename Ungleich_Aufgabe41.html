Bestimmen Sie die Lösungsmenge der
Ungleichung für x ∈ ℝ: 

x² - 8x + 12 ≥ 0 

(x - 6)(x - 2) ≥ 0 

(x - 6)(x - 2) ist größer oder gleich 0, wenn 

x - 6 ≥ 0 --> x ≥ 6 

x - 2 ≥ 0 --> x ≥ 2 

L1 = x ≥ 6 ∩ x ≥ 2 = x ≥ 6 

oder wenn 

x - 6 ≤ 0 --> x ≤ 6 

x - 2 ≤ 0 --> x ≤ 2 

L2 = x ≤ 6 ∩ x ≤ 2 = x ≤ 2 

L = L1 ∪ L2 = x ≥ 6 ∪ x ≤ 2 

L = x ≥ 6 ∪ x ≤ 2

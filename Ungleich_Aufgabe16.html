Bestimmen Sie die Lösungsmenge der
Ungleichung für x ∈ ℝ: 

   x         x  
------- - ------- < 0        
 x + 1     x - 1  

x ≠ ± 1, weil sonst d1e Nenner x + 1 = 0 
oder x - 1 = 0. Division durch 0 nicht 
definiert in ℝ. 

Fallunterscheidung, Hauptnenner (x + 1)(x - 1): 

1. Fall (x +1)(x - 1) > 0, 

dies ist dann der Fall,wenn 

(x + 1) > 0 --> x > -1 und 

(x - 1) > 0 --> x > 1 

x > -1 ∩ x > 1 = 1. Definitionsmenge für den 1. Fall D1 = 1 < x < ∞   

oder wenn

(x + 1) < 0 --> x < -1 und 

(x - 1) < 0 --> x < 1 

x < - 1 ∩ x < 1 = 2. Definitionsmenge für den 1. Fall D2 = -∞ < x < -1

   x         x  
------- - ------- < 0 |*(x + 1)(x - 1)   
 x + 1     x - 1 

x(x - 1) - x(x + 1) < 0 

x² - x - x² - x < 0 

-2x < 0 |+2x 

2x > 0 |:2 

x > 0 

x > 0 ∩ D1 = L1 = 1 < x < ∞ 

x > 0 ∩ D2 = L2 = ∅  

L1gesamt = 1 < x < ∞ 

2. Fall (x +1)(x - 1) < 0, 

ist dann der Fall, wenn 

(x + 1) > 0 --> x > -1 und 

(x - 1) < 0 --> x < 1 

x > - 1 ∩ x < 1 = 1. Definitionsmenge für den 2. Fall D3 = -1 < x < 1

oder wenn

(x + 1) < 0 --> x < -1 und 

(x - 1) < 0 --> x > 1 

x < - 1 ∩ x > 1 = 2. Definitionsmenge für den 2. Fall D4= ∅
